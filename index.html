Workflow Core

Workflow Core 是一个轻量级的工作流引擎，以 .NET Standard为目标框架。可以满足多任务下长流程一般都需要状态跟踪的，它支持插件化的持久性和并发提供程序，以允许多节点集群。

安装

安装 Nuget 包 “WorkflowCore”

使用nuget

PM> Install-Package WorkflowCore

使用 .net cli

dotnet add package WorkflowCore

Fluent API

使用fluent API定义工作流

public class MyWorkflow : IWorkflow
{
    public void Build(IWorkflowBuilder<MyData> builder)
    {    
        builder
           .StartWith<Task1>()
           .Then<Task2>()
           .Then<Task3>;
    }
}
